# Git Hooks Setup Plan

## Scope

- Add a committed pre-commit hook under `.githooks/`.
- Add `just setup` as the canonical hook activation command.
- Document hook setup as the first required clone step.
- Verify setup is idempotent and unformatted staged Rust changes are rejected.

## Commands

- `just setup`
- `cargo fmt --all -- --check`
- `cargo clippy --workspace --all-targets -- -D warnings`
- `cargo test --workspace`
- `cargo build --workspace`
